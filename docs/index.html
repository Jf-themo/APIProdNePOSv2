API NeoApp Producción V2

Descripción: API RESTful para la gestión de usuarios, autenticación, facturación y notas crédito.
Tecnologías: TypeScript, Express, MySQL, Axios.
Documentación generada con JSDoc.

Guía rápida de uso

  • Instalación: npm install
  • Configuración: Edita el archivo .env y src/database.ts
  • Ejecución: npm run build y npm start
  • Documentación: Explora las clases y métodos en este sitio.

Principales procesos documentados

  • Autenticación: Obtención y validación de token de acceso.
  • Gestión de usuarios: CRUD de usuarios y autenticación.
  • Facturación: Creación, consulta y eliminación de facturas.
  • Notas crédito: Creación de notas crédito.

Clases principales

  • People: Clase base para personas.
  • User: Extiende People, añade atributos y métodos de usuario.

Rutas principales

  • /list-all-user - Lista usuarios
  • /create-user - Crea usuario
  • /sing-in - Inicio de sesión
  • /get-token - Obtiene token
  • /create-bill - Crea factura
  • /create-note-credit - Crea nota crédito
